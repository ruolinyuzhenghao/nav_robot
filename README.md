# nav_robot
## 介绍
这是一个用于小车或双体无人船自主导航的项目！  
功能模块有全覆盖路径规划、多目标点巡航与视觉追踪、上下位机串口通信以及水面船与水下机器人协同导航。
## 环境
+ ubutnu 20.04
+ Ros noetic
+ opencv 4.6.0
+ python 3.8
